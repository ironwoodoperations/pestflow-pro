PART F — SHARED ANALYTICS HOOKS (src/hooks/)
================================================================================

useGa4Runs.ts (85 ln) — table: ga4_runs
  Returns: { loading, running, error, latestRun, runCheck }
  latestRun.data: { total_users, total_sessions, avg_engagement_rate,
    total_page_views, channels[], top_pages[] }
  Consumers: Ga4AnalyticsTile.

useGscRuns.ts (~84 ln) — table: gsc_runs
  Returns: { loading, running, error, latestRun, runCheck }
  latestRun.data: { total_clicks, total_impressions, avg_ctr, avg_position }
  Consumers: GscAnalyticsTile.

usePageSpeedRuns.ts (~85 ln) — table: pagespeed_runs
  Returns: { loading, running, error, latestRun, runCheck }
  latestRun.data: { scores: { performance, accessibility, best-practices, seo }, ... }
  Consumers: SitePerformanceTile, SeoOverviewTab (PageSpeedMini).

useSeoRuns.ts (~152 ln) — table: seo_runs (3 kinds: rankings / competitors / opportunities)
  Returns: { rankings, competitors, opportunities } where each is a
    SeoKindState { data, loading, error, lastRunAt } + per-kind runNow().
  Consumers: SeoAnalyticsTile, DashboardSeoWidget.

useZernioRuns.ts (~84 ln) — table: zernio_runs
  Returns: { latestRun, loading, running, error, runCheck }
  latestRun.data: platform-keyed { youtube|facebook|instagram: { followers,
    engagement, reach } }. followers = NULL on all platforms (S229 backlog).
  Consumers: SocialAnalyticsTile (reports), DashboardSocialWidget.

useGoogleAnalytics.ts (~44 ln) — GA4 page-view tracking script (S231 Phase 1).
  NOT a data-read hook; loads gtag.js. Unrelated to analytics tiles' reads.

F2. TIER GATING — INSIGHTS TAB UNGATED FOR Gsc/Ga4 TILES
