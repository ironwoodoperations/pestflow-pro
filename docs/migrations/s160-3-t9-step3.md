## CHECK constraint installation

Migration: `branding_no_legacy_template_constraint`

```sql
ALTER TABLE settings
ADD CONSTRAINT branding_no_legacy_template
CHECK (
  key != 'branding'
  OR value IS NULL
  OR jsonb_typeof(value) != 'object'
  OR NOT (value ? 'template')
);
```
