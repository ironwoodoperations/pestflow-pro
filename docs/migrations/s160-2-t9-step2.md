## UPDATE executed

```sql
UPDATE settings
SET value = 
  (value - 'template') || 
  jsonb_build_object(
    'theme', 
    COALESCE(value->'theme', value->'template')
  )
WHERE key = 'branding'
  AND jsonb_typeof(value) = 'object'
  AND value ? 'template';
```

COALESCE prefers existing `theme` (from dual-write saves) over `template` ‚Äî all 5 rows had only `template`, so value was promoted directly.
