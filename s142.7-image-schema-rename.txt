## TASK 2: Data copy — populate new columns from old

Run each SQL block via mcp__claude_ai_Supabase__execute_sql. Report ALL counts before proceeding to Task 3.

BLOCK 2a — copy page_content.image_url → page_hero_image_url:
  UPDATE page_content
  SET page_hero_image_url = image_url
  WHERE image_url IS NOT NULL AND image_url != '' AND page_hero_image_url IS NULL;

  SELECT COUNT(*) AS rows_with_page_hero_image_url FROM page_content WHERE page_hero_image_url IS NOT NULL;

BLOCK 2b — copy image_urls array to individual columns:
  UPDATE page_content
  SET
    image_1_url = NULLIF(image_urls->>0, ''),
    image_2_url = NULLIF(image_urls->>1, ''),
    image_3_url = NULLIF(image_urls->>2, '')
  WHERE image_urls IS NOT NULL AND jsonb_array_length(image_urls) > 0;

  SELECT
    COUNT(*) FILTER (WHERE image_1_url IS NOT NULL) AS rows_with_image_1,
    COUNT(*) FILTER (WHERE image_2_url IS NOT NULL) AS rows_with_image_2,
    COUNT(*) FILTER (WHERE image_3_url IS NOT NULL) AS rows_with_image_3
  FROM page_content;

BLOCK 2c — copy settings.hero_media.image_url → master_hero_image_url key:
  UPDATE settings
  SET value = value || jsonb_build_object('master_hero_image_url', value->>'image_url')
  WHERE key = 'hero_media'
    AND (value->>'image_url') IS NOT NULL
    AND value->>'image_url' != ''
    AND (value->'master_hero_image_url') IS NULL;

  SELECT tenant_id,
    LEFT(value->>'image_url', 60) AS old_key,
    LEFT(value->>'master_hero_image_url', 60) AS new_key
  FROM settings WHERE key = 'hero_media' AND (value->>'image_url') IS NOT NULL;

BLOCK 2d — Dang verification (tenant_id = 1611b16f-381b-4d4f-ba3a-fbde56ad425b):
  SELECT
    page_slug,
    LEFT(COALESCE(image_url,''), 55) AS old_hero,
    LEFT(COALESCE(page_hero_image_url,''), 55) AS new_hero,
    jsonb_array_length(COALESCE(image_urls, '[]'::jsonb)) AS old_img_count,
    (CASE WHEN image_1_url IS NOT NULL THEN 1 ELSE 0 END
     + CASE WHEN image_2_url IS NOT NULL THEN 1 ELSE 0 END
     + CASE WHEN image_3_url IS NOT NULL THEN 1 ELSE 0 END) AS new_img_count
  FROM page_content
  WHERE tenant_id = '1611b16f-381b-4d4f-ba3a-fbde56ad425b'
  ORDER BY page_slug;

STOP RULE: if any Dang row has old_hero set but new_hero is NULL — stop and report before continuing.
